# IHK Projektdokumentation

## Projekt

Implementierung einer skalierbaren hybriden
Netzwerkarchitektur in der Google Cloud
mit Network Connectivity Center.

## Ausbildungsbetrieb

Söldner Consult GmbH

## Technologien

- Google Cloud Platform
- Network Connectivity Center
- Cloud VPN (HA VPN)
- Cloud Router
- VPC Networks
- Cloud Firewall
- Cloud Monitoring

## Struktur

- frontmatter → Deckblatt und Verzeichnisse
- chapters → Hauptkapitel der Dokumentation
- appendix → Anhänge und technische Dokumentation
- bibliography → Quellenverzeichnis
- images → Diagramme und Screenshots

## Build

Die Dokumentation kann lokal oder mit Overleaf kompiliert werden.
