\noindent\textbf{Namenskonventionen für das Scripting (Variablen)} \\
Zur Automatisierung des Deployments wurden spezifische Namenskonventionen und Präfixe für die Cloud-Ressourcen definiert. Die verwendeten Kürzel sind im folgenden Block aufgeschlüsselt:

\begin{lstlisting}[language=bash, caption={Ressourcen-Kürzel und Namenskonventionen}]
# --- Definition der Ressourcen ---
# ASN  : Autonomous System Number
# BR   : Branch (Lokale Zweigstelle / On-Premises)
# CIDR : Classless Inter-Domain Routing (IP-Adressbereich)
# CR   : Cloud Router
# EU   : Europe (Bezeichnet die GCP-Region europe-west3)
# GW   : Gateway (HA-VPN)
# TUN  : Tunnel (IPsec)
\end{lstlisting}
